# Visualização Hidrográfica do Rio Grande do Sul

Mapa interativo de alta densidade da malha hidrográfica do RS, processando dados vetoriais do IBGE com otimização para web.

**[Acesse:](https://jordanozdanski.github.io/mapa-hidrografia-rs/)**


* **Dados:** IBGE (BC250) e Esri World Imagery

## Detalhes:
* **Otimização:** Filtragem de segmentos de drenagem irrelevantes (< 0.015°) e simplificação geométrica.
